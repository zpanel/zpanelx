git-verify-tag(1)
=================

NAME
----
git-verify-tag - Check the GPG signature of tags

SYNOPSIS
--------
'git-verify-tag' <tag>...

DESCRIPTION
-----------
Validates the gpg signature created by git-tag.

OPTIONS
-------
<tag>::
	SHA1 identifier of a git tag object.
