選擇導入選項 

要導入地址簿，你需要的日期將在CSV文件格式.主張的CSV逗號分隔值，並且將類似於：

名字，姓氏，姓名， 電郵地址，街道，城市,....

如果你輸入你的地址簿從另一個電子郵件用戶端，如Outlook ，你可以匯出你的地址數據為CSV此計劃內.

例如，如果你想導入地址簿從Outlook Express ，你將採取下列步驟.

    1 ：在Outlook中，訪問文件->“導出”通訊錄. In Outlook, visit File --> Export --> Address Book.

    2 ：你將可選擇轉換出口的格式.will then have the option to select what sort of format you would like the export to convert to.

    選擇“文本文件（逗號分隔值） ”

    3 ：然後，你可以選擇儲存的檔案在你的電腦某個.選擇瀏覽器按鈕，找到一個合適的地點和名稱的文件.

    4 ：現在，你可以選擇其中的數據地址簿你想導入到你的郵箱. 點擊“完成”當你的工作要做.

    5 ：你的地址簿的CSV現在應該順利完成.現在返回到郵局界面.

    訪問地址簿功能，並選擇下面的按鈕“輸入入” .填寫必要的資料.

第一列包含的CSV欄描述： 大多數CSV文件，其中包括新創建的文件通過 Outlook中包含欄說明.如果你不知道，你就可以檢查開放在記事本文件的CSV標題 First row contains CSV column description: Most CSV files, including files created via Outlook, contain the column descriptions. If your not sure, you can check if your CSV contains the headings by opening the file in Notepad.

點擊“瀏覽”按鈕，從地址匯入書，並選擇你的檔案創建.然後點擊“導入文件”和你幾乎沒有！

選擇相應的領域 [top]

以下你可以選擇哪些領域，從你的CSV比賽的郵箱地址簿領域.Selecting corresponding fields [top]

Here you can select which fields from your CSV match with the WebMail address book fields.

郵局將試圖找到最佳的比賽，但如果它不正確，你可以改變它通過每個領域的下拉菜單.

如果有外地的“選擇”這意味著，郵局無法找到數據區域.這可能是因為它不包括在你的CSV .它不會影響出口或造成任何問題，只要電子郵件地址字段是正確連接起來.

一旦你這樣做，單擊“導出” .

如果成功的話，你將會收到通知的一些新的接觸，然後向你的新最新的地址簿.
