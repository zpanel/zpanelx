Sync Utility für Outlook [ Hilfe Index ]

Was ist das $pref['brandname'] Sync Utility?

Dieses Utility erlaubt es komfortabel, Ihr WebMailkonto mit Outlook zu synchronisieren. Es ist möglich alle Adressbucheinträge in Outlook zu kopieren und von Outlook in das WebMail.

Wie synchronisiere ich meine Mails mit Outlook?

Um WebMail mit Outlook zu synchronisieren, gehen Sie wie folgt vor:

  1. Zuerst müssen Sie das @Mail Sync Installation Utility von folgendem Link herunterladen: http://atmail.com/websync/websync-53.exe
  2. In Microsoft Outlook, klicken Sie auf Tools > Optionen
  1. Dann ändern Sie die Konfiguration im Mail Sync Utility Panel. Klicken Sie auf Tools > Optionen, dann wählen Sie den Reiter @Mail Sync Konfiguration"


  1. Speichern Sie alle Änderungen und verlassen Sie das Konteneinstellungsfenster.

Wie synchronisiere ich meinen Webmail Postfach mit Outlook?

Nachdem Sie die Mailsynchronisation aktiviert haben (siehe oben), sollten Sie einen Button "@Mail synchronisieren" in ihrer Werkzeugleiste sehen:


Mit klick auf die Schaltfläche synchronisiert sich Outlook mit Ihrem WebMail Postfach automatisch.
