Addressbuch FAQ [ Help Index ]

Wie kann ich mein bestehendes Adressbuch importieren?

Um ein Adressbuch zu importieren, handeln Sie wie folgt:

  • Exportieren Sie das Adressbuch eines anderen E-Mailclients (Outlook, Outlook Express, ...) als CSV-Datei
  • Importieren Sie das Adressbuch in @Mail unter: Addressbuch > Import
  • Definieren Sie die Felder für das Adressbuch

Wie kann ich mein Adressbuch exportieren?

Um ein Adressbuch zu exportieren genügt es, auf "Exportieren" im Adressbuch zu klicken. Es wird dann als CSV Datei gespeichert.

Wie erstelle ich meine eigene, persönliche Gruppe?

Um Ihre eigene Persönliche Gruppe zu erstellen, klicken Sie einfach auf den "Neue Gruppe" Button im Adressbuchmenü.
