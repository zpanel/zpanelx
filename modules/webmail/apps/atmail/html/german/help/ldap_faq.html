LDAP FAQ [ Help Index ]

Wie frage ich einen LDAP-Server ab?

Um einen LDAP-Server abzufragen, gehen Sie wie folgt vor:

  1. Wechseln Sie nach > Addressbuch
  2. Wählen Sie im "Suchtyp"-Dropdownfeld "LDAP-Server"
  3. Tragen Sie die Details zu Ihrem LDAP-Server unten ein
  4. Füllen Sie die Suchfelder aus und klicken anschließend auf "Datenbank abfragen"

Wie lege ich einen externen LDAP-Server fest?

Um einen externen LDAP-Server anzugeben, füllen Sie das Feld im "LDAP Server" Fenster aus. Füllen Sie die Felder wie in einer normalen Abfrage aus, gefolgt von einem Klick auf "Datenbank abfragen"
