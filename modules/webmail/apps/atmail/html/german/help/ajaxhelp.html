AJAX Interface [ Help Index ]

Was ist die AJAX Oberfläche?

Bei der AJAX Oberfläche handelt es sich um eine auf Javascript basierende Version der einfachen Oberfläche. Durch asynchrone Verarbeitung ist Sie deutlich schneller und performanter als die herkömlicheOHTML-Oberflächen.

Einer der vielen Vorteile der AJAX Oberfläche ist die Zwischenspeicherung im Browser. Dadurch entfällt das Neuladen der Seite.

Wie nutze ich die AJAX Oberfläche?

Die AJAX Oberfläche wird automatisch aktiviert wenn Sie die entsprechende Option bei der Anmeldung auswählen.

Wie nutze ich die AJAX Oberfläche?

Ueber die AJAX Oberfläche stehen Ihnen die folgenden Funktionen zur Verfügung:

  • Mittels Shift+Klicken oder Strg+Klicken/Ctrl+Klicken können mehrere Nachrichten gleichzeitig selektiert werden.



  • Drücken Sie die 'Entf'/'Delete' Taste um selektierte Nachrichten zu löschen.
  • Use the top mail-headers to sort emails within the browser, without having to request the messages from the mail-server again.
  • Alle bereits gelesenen Nachrichten werden im Browser zwischengespeichert. Dadurch entfällt das Neuladen der Seite wenn Sie dieselbe Nachricht erneut ansehen möchten.
  • Beim Antworten auf eine Nachricht öffnet sich automatisch ein Ppopup-Fenster. Dadurch entfällt das Neuladen der Seite.

  • Weitere Optionen erhalten Sie, wenn Sie auf selektierten Nachrichten Rechtsklicken.

  • Mit den Tasten 'Auf' und 'Ab' können Sie zwischen den Nachrichten hin- und herschalten.

  • Um Nachrichten zu verschieben, selektieren Sie diese und ziehen Sie die Nachrichten in den gewünschten Ordner.

     

  • Die Anzeigespalten innerhalb des Postfachs und des Adressbuchs können automatisch angepasst werden. Klicken Sie dazu auf das Symbol in der rechten Ecke um die Anzeigeoptionen anzupassen.
