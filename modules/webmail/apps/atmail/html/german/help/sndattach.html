Wie kann ich Dateien in meine Nachrichten einfügen?

Um Dateien anzuhängen, gehen Sie wie folgt vor:

  1. Klicken Sie zuerst auf den "Anhängen"-Button im E-Mail erstellen Fenster.
  2. Um Attachments hinzuzufügen wird ein neues Fenster geöffnet; klicken Sie auf den "Browse"-Button, um eine Datei auf Ihrer Festplatte auszuwählen
  3. Klicken Sie auf "OK" wenn Sie den passenden Anhang ausgewählt haben.
  4. Sobald ein Dateiupload fertig ist, können Sie die Größe des Anhanges überprüfen oder den Anhang auch wieder löschen.

Die Anhänge müssen in ein anderes Format konvertiert werden, damit andere E-Mailprogramme sie lesen können. Der Konvertiervorgang vergrößert den Dateianhang um ca. 33%, sodaß eine 2MB große Datei auf ca. 2,66MB anwächst.
