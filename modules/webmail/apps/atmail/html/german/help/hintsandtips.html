WebMail Tipps und Tricks
  • Erhöhen Sie die Performance des WebMail Systems, indem Sie Ihren Browser veranlassen, HTML und Bilder zu cachen. Im Internet Explorer ist diese Option zu finden unter: Internetoptionen -> Temporary Internet Files -> Einstellungen -> Neuere Version gespeicherter Seiten suchen: Automatisch.
  • Indem Sie auf eine Nachtricht Rechtsklicken, öffnet sich ein Kontextmenü. Dieses offeriert Ihnen einen komfortablen Weg, auf Nachrichten zu antworten, sie weiterzuleiten oder eine Nachricht zu öffnen.
  • Über den "Empfänger hinzufügen" Knopf können Sie E-Mail Adressen aus Ihrem Adressbuch in das Empfängerfeld einfügen. Wenn Sie auf den Knopf klicken öffnet sich hierzu das Adressbuch.
  • Wollen Sie Adressen aus anderen LDAP-Servern verwenden? Kein Problem - im Nach Personen suchen Fenster können Sie den LDAP-Server, sowie die Suchkriterien angeben.
  • Wussten Sie, daß Sie Nachrichten auf einfache Art und Weise per Drag n Drop von einem Ordner in einen anderen verschieben können?
