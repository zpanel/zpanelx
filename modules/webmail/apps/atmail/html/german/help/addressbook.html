Adressbuch [ Hilfe Index ]

Benutzen Sie Ihr Adressbuch um Kontaktinformationen von Ihren Freunden und Partner zu speichern. Auf diesem Weg können Sie jederzeit auf diese Adressen zugreifen wenn Sie z.B. eine E-Mail verfassen.

Klicken Sie auf 'Eintrag hinzufügen' um einen neuen Eintrag ins Adressbuch hinzuzufügen. Bitte geben Sie die Benutzerdeatails ein um diese im Adressbuch zu speichern. Sie können die Einträge ändern und löschen wenn Sie auf das entsprechende Symbol in der Navigation klicken.
