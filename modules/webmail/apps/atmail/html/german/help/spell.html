Rechtschreibprüfung [ Hilfe Index ]

Mit dem Rechtschreibprüfungs Tool können Sie Ihre Nachrichten nach Rechtschreibfehlern durchsuchen. Wenn ein Wort nicht in der Datenbank gefunden werden kann, wird eine Seite mit Vorschlägen angezeigt. Wählen Sie ein Wort aus der Liste und klicken Sie auf "Ändern". Wenn ein Wort nicht in der Liste steht, können Sie es in das Wörterbuch eintragen lassen.

Wenn ein Wort richtig geschrieben ist, das Tool jedoch einen Fehler zeigt, klicken Sie auf "Ins Wörterbuch eintragen" um das Wort in das Wörterbuch hinzuzufügen. Um den Fehler zu ignorieren, klicken Sie auf "Ignorieren".

Das Verfassen Fenster wird wieder angezeigt, wenn keine Fehler gefunden wurden.
