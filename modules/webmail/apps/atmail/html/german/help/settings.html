Einstellungen ändern [ Hilfe Index ]

Verwenden Sie die Eigenschaften Option um die Eigenschaften Ihres E-Mail Accounts anzupassen. Passen Sie die Oberfläche auf Ihre Bedürfnisse an, die Eigenschaften beinhalten die folgenden Einstellungen:

WebMail Einstellungen

  • Vollständiger Name
    Dieser wird bei ausgehenden E-Mails als Absendernamen verwendet.
  • Antwortadresse
    Geben Sie die Antwortadresse ein, wenn diese Ihrem Benutzernamen nicht entspricht.
  • E-Mails sortieren nach
    Wählen Sie ein Sortierungskriterium aus, anhand dessen Ihre Emails sortiert werden sollen.
  • Layout
    Ändern Sie die Oberfläche des WebMails und wählen Sie die einfache Oberfläche für ältere Browser aus.
  • Sprache
    Wählen Sie die Sprache der Oberfläche aus.
  • Nachrichten anzeigen
    Wählen Sie die Anzahl von E-Mails die angezeigt werden im Eingangsordner.
  • Posteingangs Aktualisierungsintervall
    Geben Sie den Intervall (in Minuten) an, in welchem der Posteingang aktualisiert werden soll. Sie werden durch einen Hinweiston auf neue Nachrichten aufmerksam gemacht.
  • Zeitformat
    Wählen Sie die Voreinstellung, um im WebMail und im Kalender die Zeit anzuzeigen
  • Datumsformat
    Wählen Sie das Datumsformat das angezeigt werden soll
  • Zeitzone wählen
    Wählen Sie die Zeitzone aus in der Sie sich befinden. Diese wird bei E-Mails immer angegeben.
  • E-Mail Signatur
    Geben Sie hier eine E-Mail Signatur an, die bei allen ausgehenden E-Mails angehängt wird.
  • Papierkorb leeren bei der Abmeldung
    Wählen Sie diese Option aus, wenn beim Abmelden der Papierkorb geleert werden soll.
  • HTML Editor
    Wählen Sie diese Option, wenn Sie einen HTML Editor verwenden möchten.
  • E-Mail Signatur
    Erstellen Sie Ihre Signatur. Sie wird an alle ausgehenden Nachrichten automatisch angehängt
