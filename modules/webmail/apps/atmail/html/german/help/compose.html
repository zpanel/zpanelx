E-Mail senden [nach oben]
 Über Ihren WebMail Account können Sie E-Mails an jede beliebige aktive E-Mail Adresse versenden. Mit dem HTML-Editor (nur IE4/IE5) können Sie HTML E-Mails verfassen, Sie können jedoch auch den Nur-Text Editor verwenden um Ihre E-Mails zu verfassen.

Wie versende ich E-Mails über die WebMail Oberfläche

  1. Loggen Sie sich in Ihren Account ein.
  2. Klicken Sie auf E-Mail verfassen in der Navigation
  3. Füllen Sie die Kopfzeilen Informationen aus:
    • An: Geben Sie die Empfänger Adresse ein, oder wählen Sie eine Adresse aus dem Adressbuch aus. Wenn Sie die E-Mail an mehrere Adressen versenden wollen, verwenden Sie "," zum Separieren der E-Mail Adressen.
    • Betreff: Geben Sie den Betreff der Nachricht ein.
    • CC: Geben Sie eine weitere E-Mail Adresse ein. Dieser Empfänger erhält eine Kopie der Nachricht.
    • BCC: Geben Sie die E-Mail Adresse eines weiteren Empfängers ein der ebenfalls eine Kopie erhalten soll, jedoch ohne das die erste Person davon erfährt.
  4. Verfassen Sie Ihre Nachricht im entsprechendem Textfeld.
  5. Um einen Anhang an die E-Mail anzubinden, klicken Sie auf den "Anhänge" Knopf. Es öffnet sich nun ein Fenster, über welches Sie Anhänge laden können.
  6. Wenn Sie die E-Mail fertig erstellt haben, klicken Sie auf E-Mail senden.

Funktionen von ausgehenden E-Mails [nach oben]

  • Anhänge
    Um Dateien den Email Nachrichten anzuhängen klicken Sie zuerst auf Anhängen. Ein neues Fenster erscheint um Anhänge hinzuzufügen; klicken Sie auf "Durchsuchen" und wählen Sie die Datei von Ihrer Festplatte aus. Klicken SIe auf "OK" wenn Sie die Anhänge gewählt haben. Sobald die Dateien hochgeladen sind können Sie noch die Größe überprüfen und bei Bedarf löschen. Mehrere Anhänge sind ebenfalls möglich; aus Sicherheitsgründen können die Anhänge maximal <2 MByte.

  • Signaturen
    Wenn Sie Ihre Email Nachricht senden, können Sie eine Signatur unten zu jeder Email einbauen. Beachten Sie die Eigenschaften Konsole für mehr Informationen.
