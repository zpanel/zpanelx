Wie blockiere ich E-Mails von bestimmten Absendern?

  • Wenn Sie eine E-Mailadresse blockieren, sind Sie von dieser E-Mailadresse aus nicht mehr zu erreichen. Um eine Adresse zu blockieren gehen Sie wie folgt vor:

    1. Wechseln Sie zur "Blockiere E-Mailadresse/Spammer" Sektion in der Ordneransicht.
    2. Geben Sie die Adresse im "Geben Sie zu blockierende E-Mailadresse ein"-Feld ein.
    3. Klicken Sie auf "E-Mailadresse hinzufügen".

  • Die Adresse scheint dann in der Liste der blockierten Adresse auf. Sie können auch E-Mails anhand des Betreffs blockieren, die Vorgehensweise ist die selbe.

Was ist die maximale Größe einer empfangenen Nachricht?

Mit einem $pref['brandname'] E-Mail Postfach können Sie Nachrichten bis zu $pref['max_msg_size']MB versenden und empfangen. Die endgültige Größe der Nachricht, die Sie versenden, kann also nicht größer als $pref['max_msg_size'] MB sein.

NOTIZ: Wenn Sie einer Nachricht eine Datei anhängen, so muß diese kodiert werden, was sich wiederum negativ auf die Nachrichtengröße auswirkt. Anhänge, die per E-Mail versendet werden, müssen in eine anderes Format konvertiert werden, damit andere E-Mailprogramme wie Outlook es lesen können. Diese konvertierung vergrößert die Originaldatei um ca. 33%, sodaß eine 2MB große Datei auf 2.66MB anwächst.
