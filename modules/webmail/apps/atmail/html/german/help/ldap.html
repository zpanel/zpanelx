LDAP Suchmaschine [ Hilfe Index ]

Benutzen Sie die LDAP Suchmaschine um Benutzeremailadressen, Telefonnummern und Postadressen zu finden. LDAP ist eine öffentliche Datenbank mit Benutzerdetails aus der ganzen Welt. Für die Suche geben Sie bitte den Namen der gesuchten Person ein.

Wenn Sie sich die Resultate anschauen, können Sie die Benutzer in Ihr Adressbuch hinzufügen, direkt eine Email verfassen oder mehr Informationen zu der Person erhalten.

Wenn die Funktion aktiviert ist können Sie nach Benutzer eines bestimmten Providers suchen indem Sie auf 'Suchenart' klicken und dort auf 'Dursuchen der System Datenbank' gehen.
