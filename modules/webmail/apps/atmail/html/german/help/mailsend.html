Kann ich eine Nachricht an mehr als eine Person senden?

  • Sie können die Nachrichten an 100 Personen zugleich senden.
  • Um eine Nachricht an mehr als eine Person zu senden, fügen Sie zusätzliche E-Mail Adressen in die Felder: An:, Cc:, oder Bcc ein. Die Adressen werden separiert durch ";" (z.b.: benlee@isp.net; john@mycompany.com; info@atmail.com usw...).

Wie überprüfe ich die Rechtschreibung einer Nachricht?

  • Um die Rechtschreibung Ihres Textes zu überprüfen, klicken Sie auf "Rechtschreibeüberprüfung", gleich über dem Empfängerfeld in Ihrem "E-Mail verfassen" Fenster.
  • Falls ein Wort aus Ihrer Nachricht nicht von der Rechtschreibprüfung erkannt wird, erscheint es in einer kleinen Box. Falls möglich schlägt Ihnen das Wörtebuch Alternativen vor, die Sie mit einem Klick auf das gewünschte Wort und auf den "Ändern" Button automatisch einfügen lassen können. Sie können auch einfach das angezeigte Wort überschreiben. Sobald Sie fertig sind, genügt ein Klick auf "OK" und Ihre Änderungen werden gespeichert. Ein Klick auf "Abbrechen" verwirft alle von Ihnen vorgenommenen Änderungen.

Kann ich unfertige Mails als Entwurf speichern, ohne Sie zu senden?

Ja. Um eine unfertige E-Mail zu speichern ohne sie zu versenden, klicken Sie auf den "Nachricht speichern" Button nahe des oberen Randes des "Nachricht verfassen" Fensters. Nachdem die Nachricht gespeichert wurde kann sie jederzeit wieder geöffnet werden, indem Sie wie folgt vorgehen:

  1. klicken auf den "Entwürfe"-Link auf dem linken Navigationsmenü.
  2. Klicken Sie auf die Nachricht mit der Sie fortsetzen wollen.

Entwürde werden solange gespeichert bis Sie sie löschen.

Was ist die maximale Größe einer zu versendenden Nachricht?

Mit einem $pref['brandname'] Konto können Sie Nachrichten bis zu einer Größe von $pref['max_msg_size'] MB senden und empfangen. Die Höchstgrenze für E-Mails mit kodierten Anhängen darf $pref['max_msg_size'] MB nicht überschreiten.

ANMERKUNG: Die Anhänge müssen in ein anderes Format konvertiert werden, damit andere E-Mailprogramme sie lesen können. Der Konvertiervorgang vergrößert den Dateianhang um ca. 33%, sodaß eine 2MB große Datei auf ca. 2,66MB anwächst.

Warum bekomme ich eine "Übertragung fehlgeschlagen"-Meldung?

Normalerweise bekommen Sie eine "Übertragung Fehlgeschlagen" Nachricht, wenn Sie eine inkorrekte oder ungültige E-Mailadresse eingegeben haben. In der Regel bekommen Sie eine Mail von "Mail Delivery Subsystem" oder "MAILER-DAEMON" mit einem Betreff wie "failure notice" o.ä. Dies bedeuted, dass eine übermittlung Ihrer Nachricht nicht möglich war und sie deswegen wieder zu Ihnen zurückgeschickt wurde. Die Nachrichten werden vom verarbeitenden Mailserver automatisch generiert und enthalten eine kurze Fehlermeldung.
