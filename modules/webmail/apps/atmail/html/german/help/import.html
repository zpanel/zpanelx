Adressbuchimport Hilfe [ Hilfe Index ]

Die hier gefundene Information soll Ihnen helfen, ein existierendes Adressbuch in Ihr WebMailkonto zu importieren.

  • Importieroptionen einstellen
  • Wählen Sie die entsprechenden Felder aus

Importoptionen auswählen 

Um ein Adressbuch zu importieren, muß das Datum im CSV Format vorliegen. CSV steht für Comma Separated Values, und sieht in etwas so aus:

Vorname,Nachname,Name,E-mailadresses,Adresse,Stadt,....

Wenn Sie Ihr Adressbuch aus einem anderen E-Mailprogramm heraus importieren, können Sie Ihre Daten auch als CSV Speichern.

Falls sie z.b.: ein Adressbuch aus Outlook Express importieren wollen, gehen Sie wie folgt vor.

    1: In Outlook, klicken sie auf Datei -> Export -> Adressbuch

    2: Wählen Sie nur das Zielformat des Exports aus.

    Wählen Sie "Textfile (Comma Separated Value" aus

    3: Sie haben dann die Option, die generierte CSV-Datei auf Ihrem Computer zu speichern. Wählen Sie einen geeigneten Speicherort aus.

    4: Sie können jetzt auswählen, welche Daten Sie aus Ihrem Adressbuch in die WebMail Oberfläche importieren wollen. Klicken Sie auf "Fertig", sobald der Importvorgang fertiggestellt ist.

    5: Ihr Adressbuch CSV sollte jetzt erfolgreich sein. Bitte kehren Sie zum WebMailinterface zurück.

    Rufen Sie die Adressbuchfunktion auf und drücken Sie auf den "Importieren" Button. Dort können Sie die benötigten Informationen eintragen.

Die erste Zeile enthält die CSV Spaltenbezeichnun: Die meisten CSV-Dateien, auch von Outlook generierte, enthalten die Spaltenbezeichnungen. Falls Sie nicht sicher sind, können Sie die CSV-Datei auch mit Notepad öffnen.

Klicken Sie auf den "Durchsuchen" Button in der Adressbuchimport-Ansicht und verweisen Sie auf ihre CSV-Datei. DAnn klicken Sie auf "Datei Importieren" und Sie haben es fast geschafft!

Wählen Sie die entsprechenden Felder  [top]

. Hier können Sie einstellen, welche Felder der CSV-Datei mit denen des WebMail Adressbuches übereinstimmen.

WebMail wird versuchen, das beste Ergebnis zu finden. Falls es nicht korrekt ist können Sie es in den Dropdownfeldern ändern.

Wenn ein Feld "Select" vorhanden ist, bedeutet das, daß WebMail die Daten für dieses Feld nicht finden konnte. Dies könnte u.a. passieren, wenn die Daten in Ihrer CSV-Datei nicht nicht präsent waren. Dies beeinträchtigt den Export nicht, solange das E-Mailadressfeld korrekt verlinkt ist.

Sobald Sie fertig sind, klicken Sie auf "Export".

Falls der Vorgang erfolgreich war, werden Sie über die Anzahl der neuen Kontakte informiert und zu Ihrem neuen Adressbuch weitergeleitet.
