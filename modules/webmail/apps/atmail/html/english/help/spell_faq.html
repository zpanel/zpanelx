Spell Check FAQ [ Help Index ]

How do I know my email is spell-checked?

    When the spell check is complete the browser will refresh the email-window with the new spelling. If no spelling errors are found the page is refreshed automatically

My message sends without notification.

    Please note that the message will not be sent until you click on the "Sent" Button. Should you experience this issues however, please contact your Web Administrator.


The spell-checker changes the words to a foreign language.

    Depending on which Language you have logged into the WebMail interface, the spell-check dictionary will default the local language. The default dictionary for the spell-check is set to English.
