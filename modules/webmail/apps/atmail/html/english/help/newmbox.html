Mailbox Options [ Help Index ]

This folder allows you to customize the Options for Your Mailbox and it's folders.

The Active Mailbox's section has a list of all your Mailbox folders, the number of messages in it, the size of the folders and the Option to Delete this folder.

The Create a new Mailbox allows you to add a new Mailbox folder.

The Block Email address/Spammers allows your to stop all email from an email address from getting through. Simply add the address that you do not want to receive email from here and click on Add Email.

The Auto-reply option will automatically send a reply to every email that you receive. You can optionally specify a message to send as an auto-reply message.

The Email to Folder Sorting option allows you to set WebMail to automatically move incoming mail from your inbox to specific folders.
