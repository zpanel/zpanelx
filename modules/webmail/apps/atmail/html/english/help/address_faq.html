Address Book FAQ [ Help Index ]

How can I import my existing addressbook?

To import an addressbook, do the following:

  • Export the addressbook from another client (Outlook, Express) as a text CSV file
  • Import the addressbook into Atmail by going to Addressbook > Import
  • Define the fields for the addressbook

How can I export my existing addressbook?

To export an addressbook, just click on "Export" on the Addressbook pane. The addressbook will be extracted as a CSV file.

How do I create my own personal group?

To create your own personal group, just click on the "New Group" button in the addressbook menu.
