Read Email [ Help Index ]

The email windows frame contains a list of messages received for your email account. Double click to read the message.

Via the preferences control panel you can choose which order to read your email in. To sort the mailbox by subject, received date, sender and ID. Each message in the mailbox can be marked to be moved to an external mailbox; this function allows you to select messages to be moved to the Trash folder or to select a message to move into an alternate mailbox away from the main Inbox.

Via the email window pane you can jump directly to the Options for the account, or move to the Folder information window to view settings for your mailboxes.
