Veelgestelde vragen - LDAP [ Help Index ]

Hoe zoek ik een LDAP server?

Om een LDAP server te zoeken, dient u het volgende te doen:

  1. Ga naar Zoeken > Adresboek
  2. Selecteer LDAP server in het Zoektype' drop-down menu.
  3. Geef uw LDAP server gegevens in.
  4. Vul de zoekvelden in en klik vervolgens op Database doorzoeken.

Hoe geef ik een externe LDAP server in?

Om een externe LDAP server in te geven, dient u het tekstveld in het LDAP server zoekvenster in te vullen. Vul de zoekvelden in, zoals u zou doen voor een gewone zoekopdracht en klik vervolgens op Database doorzoeken.
