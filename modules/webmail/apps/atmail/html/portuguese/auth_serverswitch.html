Security Error

The account you are trying to login as has triggered a security warning.
  • The account in the database is assosiated with the mail-server $var['origserver'] , while you are trying to access the remote account with the mail-server $var['server']
  • Access is locked since the assosiated mail-server address is no longer used
  • This is used as a security precaution to disallow access from unauthorized hostnames

To resolve the access error

  • Login to the new remote-mailserver with the existing password used in $pref['brandname']
  • Contact the administrator $this->admin_email to assosciate your account with the new mail-server address
