Ajuda do Corretor Ortográfico [ Help Index ]

Como eu saberei que meu e-mail está corrigido ortograficamente?

    Quando o corretor ortográfico estiver concluído, o janela dará um refresh e aplicará as devidas correções ao e-mail.

My message sends without notification.

    Uma mensagem nunca é enviada sem que o botão "ENVIAR" seja clicado. Você deve estar com problemas, por favor, contate o administrador.


O corretor ortográfico mudou as palavras para um idioma estrangeiro

    Dependendo de qual idioma você escolheu para se logar ao Webmail, o dicionário padrão desse idioma será utilizado. O dicionário padrão é configurado para Inglês.
