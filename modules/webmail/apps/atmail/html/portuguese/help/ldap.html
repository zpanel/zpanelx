Sistema de Busca LDAP [ Índice de Ajuda ]

Utilize o sistema de procura LDAP para encontar o endereço de email do contato pretendido, assim como outros elementos de contacto. LDAP é uma base de dados com os detalhes dos contatos. Para iniciar uma procura, tem que especificar palavras chave.

Quando consultar os resultados da procura pode selecionar para adicionar o contato no seu Livro de Endereços.

Se o seu serviço de email está ativo, pode procurar pelo domínio ou local na web para obter contatos 'Tipo de Procura' para procurar no sistema de 'Procura na base de dados'.
