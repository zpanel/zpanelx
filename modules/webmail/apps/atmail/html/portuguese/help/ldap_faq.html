Como eu pesquiso por um servidor LDAP?

Para pesquiser em um servidor LDAP faça o seguinte:

  1. Vá até Buscar > Livro de Endereços
  2. No menu drop-down com o nome "Tipo de Busca" selecione "LDAP Server".
  3. Especifique os detalhes do servidor LDAP abaixo
  4. Preencha os campos de busca e clique em "Procurar Database".

Como especifico um servidor LDAP externo.

Para criar um servidor LDAP externo, preencha o campo de texto, na janela de pesquisa chamada "LDAP Server". Preencha os campos que você preencheria para uma pesquisa normal e então clique em "Pesquisar Database"
