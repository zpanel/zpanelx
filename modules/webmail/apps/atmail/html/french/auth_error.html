Erreur d'authentification

Accès refusé - Votre ID de session != peut être validé dans la base de donnée
  • Vérifiez votre nom d'utilisateur et votre mot de passe
  • Vérifiez si votre navigateur supporte les cookies, sinon activez cette option dans votre navigateur
  • Contrôlez que deux sessions != sont pas actives sur votre compte - Un seul utilisateur par login est accepté

S'il vous est toujours impossible d'accéder à votre compte

  • Utilisez l'utilitairede mot de passepour récupérer votre mot de passe
  • Veuillez contacter l'administrateur $this->admin_email pour assistance
  • Créer un nouveau compte
