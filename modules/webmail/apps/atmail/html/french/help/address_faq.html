FAQ du carnet d'adresses [ Help Index ]

Comment est-ce que je peux importer mon carnet d'adresses actuel ?

Pour importer un carnet d'adresses, faites ce qui suit:

  • Exportez le carnet d'adresses d'un autre client de messagerie (Outlook, Outlook Express) sous la forme d'un fichier CSV
  • Importez le carnet d'adresses dans @Mail en allant dans Carnet d'adresses > Importer
  • Définissez les champs du carnet d'adresses

Comment puis-je exporter mon carnet d'adresses existant ?

Pour exporter un carnet d'adresses, cliquez simplement sur "Exporter" sur le panneau du carnet d'adresses. Le carnet d'adresses sera alors extrait sous la forme d'un fichier CSV.

Comment est-ce que je créé mon propre groupe personnel ?

Pour créer votre propre groupe personnel, cliquez simplement sur le bouton "Nouveau Groupe" dans le menu du carnet d'adresses.
