Comment puis-je attacher un fichier à un message ?

Pour attacher des fichiers à vos messages emails, faites ce qui suit:

  1. Tout d'abord, cliquez sur le bouton "Attacher" dans l'écran de composition des messages.
  2. Une nouvelle fenêtre popup va s'afficher pour ajouter des attachements; cliquez sur le bouton "Naviguer" pour sélectionner un fichier de votre disque dur.
  3. Cliquez "OK" quand vous avez sélectionné le fichier que vous souhaitez inclure en attachement
  4. Une fois que les fichiers ont été chargés, vous pouvez vérifier la taille globale ou choisir de détruire le(S) fichier(s).

NOTE: Quand vous attachez des fichiers dans vos emails, ceux-ci doivent être encodés, ce qui en augmentera la taille. Les attachements envoyés avec chaque email doivent être traduits dans un format différent avant que chaque programme puisse les lire. Ce type de transformation ajoutera environ 33% à la taille originelle de l'attachement. Par conséquent, suite à une traduction un attachement de 2 Mo deviendra un email faisant environ 2,66 Mo après encodage.
