Envoyer un email [Haut]
 Avec votre compte webmail vous pouvez envoyer des message à toutes les adresses e-mail actives sur Internet. Avec l'éditeur HTML intégré (IE4/IE5 uniquement) vous pouvez écrire des messages en HTML ou pour la compatibilité (message 'text') utiliser l'éditeur texte pour envoyer des messages standards.

Comment envoyer des emails par le Web

  1. Login sur votre compte.
  2. Cliquer Ecrire message sur la barre de navigation
  3. Remplissez les informations de l'entête:
    • A: Introduisez l'adresse e-mail du destinataire ou sélectionnez une adresse de votre carnet d'adresses. Pour plusieurs destinataires, séparez les adresses avec une virgule (,).
    • Sujet: Introduisez le sujet du message.
    • CC: Introduisez l'adresse d'un ou de plusieurs destinataires supplémentaires. Une copie du message leur sera envoyé.
    • BCC: Introduisez l'adresse e-mail des destinataires qui vont recevoir une copie du message à l'insu des autres destinataires.
  4. Composez votre message dans la zone de texte fournie.
  5. Si nécessaire, cliquez sur le bouton 'Fichier Attaché' pour charger des fichiers qui seront envoyés avec le message.
  6. Cliquez sur 'Envoyer un email' en haut ou en bas de votre écran.

Propriétés de l'envoi d'email [Haut]

  • Fichiers attachés
    Pour joindre des fichiers à vos messages, vous devez cliquer sur Parcourir et sélectionner un fichier depuis votre disque dur. Cliquez sur OK quand vous avez séléctionné le fichier. Dès que le fichier a été transféré sur le serveur vous pouvez vérifier sa taille et choisir de l'effacer. Plusieurs fichiers joints dans un message sont supportés, mais par rapport à la sécurité et l'espace disque sur le serveur, la taille des fichiers joints != peuvent pas dépasser <2-10 Mo. Pour connaître la taille exacte contactez l'administrateur du serveur.

  • Signatures
    Quand vous envoyez votre message vous pouvez définir une signature qui se rajoute en bas du message. Consultez le panneau de configuration pour plus d'informations.
