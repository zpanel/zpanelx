Lire les messages [ Index ]

La fenêtre email contient la liste de tous les messages qui sont stockés dans le répertoire. En déplaçant la souris sur un message particulier une fenêtre 'popup' vous affiche les 160 premiers caractères du message. Pour ouvrir le message en entier, cliquez sur le sujet.

Avec les préférences dans le panneau de configuration, vous pouvez choisir l'ordre dans lequel vous allez lire vos messages. Vous pouvez effectuer des tris par rapport au sujet, la date de réception, l'expédieur et le numéro ID. Vous pouvez sélectionner des message afin de les déplacer dans d'autres répertoires. Cette fonction vous permet également de déplacer les messages sélectionnés dans la poubelle.

A travers le panneau de fenêtre d'email, vous pouvez directement aller dans les Options du compte ou visualier les informations concernant les répertoires.
