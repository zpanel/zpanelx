Comment puis-je changer mon nom pour les messages sortants ?

Votre "nom affiché" est le nom qui apparait dans chaque email sortant que le destinataire verra comme étant l'expéditeur. Pour changer ce "nom affiché" vous devez:

  1. Cliquez sur les "Paramètres" depuis le menu principal.
  2. Entrez le nom voulu dans le champs "Nom Affiché" (DN).
  3. Cliquez pour sauvegarder les paramètres

Comment puis-je créer une signature ?

Une signature est un texte qui est ajouté automatiquement en bas de chaque message que vous envoyez. Cela peut être un moyen efficace d'indiquer votre numéro de téléphone et votre adresse dans l'email.

  1. Cliquez sur les "Paramètres" depuis le menu principal.
  2. Entrez votre signature dans le champs "Signature Email".
  3. Cliquez pour sauvegarder les paramètres.
