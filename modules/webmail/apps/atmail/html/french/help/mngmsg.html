Gestion des messages [ Index ]

Combien de messages email puis-je récupérer/conserver sur le Webmail $pref{brandname] ?

Pour connaître la limite de stockage de votre compte, cliquez sur la page de démarrage de l'interface webmail pour un résumé du quota de votre compte

Est-ce l'email dans ma poubelle compte pour ma limite de stockage ?

Oui. Les messages dans votre poubelle sont comptabilisés pour votre limite de stockage. Pour effacer de manière permanente le contenu de la poubelle, cliquez sur l'icone "Vider la poubelle" située en bas du menu sur la page "Lire le message". Une confirmation d'effacement de tous les messages vous sera demandée. Cliquez sur "Oui" ou sur "Non" pour retourner à la page précédente.

NOTE: Une fois que les messages sont vidés de la poubelle, ils sont effacés de manière permanente, et != peuvent plus être récupérés.

Comment est-ce que je sais lorsque j'atteinds mon quota de message ?

Le systeme Webmail $pref['brandname'] affichera votre quota dans la page de démarrage. Lorsque vous dépassez 90% de votre capacité de stockage, vous devriez considérer la suppression de messages ou demander plus d'espace disque

Comment est-ce que je vide la poubelle ?

Pour effacer de manière permanente le contenu de la poubelle, cliquez sur l'icone "Vider la poubelle" située en bas de la page "Lire Message". Ensuite, on vous demandera si vous êtes certain de vouloir effacer tous les messages. Cliquez "Oui" pour les effacer ou "Non" pour retourner à la page précédente.

NOTE: une fois que les messages sont détruits de la poubelle, ils sont effacés de manière permanente et != peuvent plus être récupérés.

Comment est-ce que je crée un nouveau dossier ?

Pour vous aider à organiser vos messages, vous pouvez créer autant de dossiers que vous le souhaitez. Pour créer un dossier, vous devez:

  1. Allez sur la section intitulée "Créer un nouveau dossier" dans la page du "Gestionnaire de Dossier".
  2. Indiquez un nom de dossier
  3. Cliquez sur le bouton "Créer un dossier".

Comment est-ce que je renomme un dossier ?

Cliquez avec le bouton droit de la souris sur la boite aux lettres sélectionnée sur la fenêtre du dossier:

  • Vous pouvez alors taper un nouveau nom de dossier ici;
  • Tapez sur la touche entrée pour valider les changements

Comment est-ce que je transfert des messages sur un de mes dossiers ?

Pour transférer un message à l'un de vos dossiers, vous devez :

  • Ouvrez le dossier qui contient le message à transférer.
  • Pour le message que vous souhaitez transférer, cliquez sur la case à gauche du message. Vous pouvez sélectionnez plus d'un message à la fois.
  • Sélectionnez le dossier de destination.
  • Cliquez sur le bouton "Déplacer".

Les message(s) seront transférés.

(NOTE: Certains messages != peuvent être transférés dans votre dossier des messages reçus et sur les dossiers des comptes POP3 externes. Vous n'avez pas besoin de transférer des messages à la poubelle; il vous suffit de cliquer sur la case à la gauche du message et de cliquer sur le bouton "Effacer")

Comment puis-je voir le contenu du dossier ?

Pour voir le contenu d'un dossier, cliquez sur celui que vous souhaitez ouvrir.

Comment supprimer un dossier ?

Pour effacer un dossier, vous devez d'abord effacer tous les messages qui sont à l'intérieur (seuls des dossiers vides peuvent être effacés). Pour détruire le dossier vous devez:

  • Cliquez sur le "Gestionnaire de Dossier" au niveau du menu principal.
  • Sélectionnez le dossier que vous souhaitez effacer et cliquer sur le lien "Détruire" situé en bas à droite du listing.

Le dossier sera effacé.
NOTE: vous != pouvez pas détruire les dossiers permanents de votre compte. Ces dossiers permanents sont celui des Messages reçus, des messages envoyés, des Brouillons et de la Poubelle.
