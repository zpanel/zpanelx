Guide d'aide à l'importation du carnet d'adresses [ Index ]

L'information fournie vous aidera à importer un carnet d'adresses existant dans votre compte Webmail.

  • Sélection des options d'import
  • Sélection de champs correspondants

Sélection des options d'import 

Pour importer un carnet d'adresses, vous aurez besoin que la doit soit dans le format du fichier CSV. CSV signifie Valeurs Séparées par des Virgules (Comma Separated Values), et devrait ressembler à:

Prénom, Nom, Adresse Email, Nom de la rue, ville,...

Si vous importez votre carnet d'adresses depuis un autre client de messagerie tel qu'Outlook, vous pouvez exporter vos données sur un fichier CSV par ce programme.

Par exemple, si vous souhaitez importer un carnet d'adresses depuis Outlook Express, vous allez suivre les étapes suivantes:

    1: Dans Outlook, visitez Fichier --> Export --> Carnet d'adresses.

    2: Vous aurez ensuite l'option de sélectionner le type de format que vous voulez exporter

    Sélectionnez "Fichier Texte (Valeurs Séparées par des Virgules)"

    3: Vous aurez ensuite la possibilité de conserver le fichier CSV sur votre ordinateur. Utilisez le navigateur pour trouver une place acceptable et donner un nom à votre fichier.

    4: vous pouvez maintenant sélectionner les données de votre carnet d'adresses que vous souhaiteriez importer dans votre Webmail. Cliquez "Finir" quand vous avez terminé.

    5: Votre CSV du carnet d'adresses devrait être mis en place avec succès. Maintenant, retournez à l'interface du Webmail.

    Visitez la fonction du Carnet d'Adresses et sélectionnez le bouton "Importer". Remplissez les informations nécessaires.

La première ligne contient le descriptif des colonnes du fichier CSV: La plupart des fichiers CSV, y compris ceux créés par Outlook, contiennent cette description des colonnes. Si vous n'êtes pas sûr, vous pouvez vérifier que votre fichier CSV contient cette entête en l'ouvrant avec Notepad.

Cliquez sur le bouton "Naviguer" depuis le menu d'import d'adresses et sélectionnez le fichier CSV que vous avez créé. Ensuite, cliquez sur "Importer le fichier" et vous y êtes !

Sélection des champs correspondants [top]

Ici vous pouvez sélectionner les champs de votre fichier CSV qui correspondent aux champs du carnet d'adresses du Webmail.

Le système essaiera de trouver la meilleure correspondance, toutefois s'il n'est pas correct, vous pouvez effectuer les changements à travers le menu déroulant de chaque champs.

S'il y a un champ avec "Sélectionner" cela signifie que le Webmail n'a pas été capable de trouver les données pour ce champs. Cela peut être parce qu'il n'a pas été inclus dans votre fichier CSV. Cela n'affectera pas l'export ni != causera un quelconque problème, tant que le champs de l'adresse email est correctement sélectionné.

Une fois que vous avez terminé, cliquez sur "Exporter".

Si cela réussit, vous serez averti du nombre de nouveaux contacts, ensuite vous serez redirigés vers votre nouveau carnet d'adresses.
