Utilitaire de synchronisation pour Outlook [ Index ]

Quel est l'utilitaire de synchronisation de $pref['brandname'] ?

L'utilitaire vous permet de synchroniser votre compte webmail avec Outlook. Avec cela, toute l'information du carnet d'adresses que vous avez sur le webmail est transférée sur Outlook, et réciproquement.

Comment est-ce que je peux autoriser la synchronisation de messagerie avec Outlook ?

Pour synchroniser votre compte Webmail avec Outlook, faites la chose suivante:

  1. Premièrement, téléchargez le fichier additionnel de synchronisation @Mail http://atmail.com/websync/websync-53.exe
  2. Sur votre fenêtre Outlook de Microsoft , allez sur Outils > Options
  1. Ensuite, définissez tous les changements dans le panneau de synchronisation des emails. Cliquez sur Outils > Options, ensuite allez sur l'onglet intitulé "Configuration de la synchronisation @Mail"


  1. Sauvegarder tous les changements, ensuite quittez la fenêtre des options de comptes.

Comment est-ce que je synchronise mon compte webmail avec Outlook ?

Après avoir autorisé la synchronisation d'email (voir ci-dessus), vous devriez voir un bouton portant le label "Synchro @Mail" sur votre barre d'outil:


Cliquez sur le bouton, et Outlook se synchronisera automatiquement avec votre compte webmail.
