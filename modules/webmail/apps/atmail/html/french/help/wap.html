Accédez à votre email en utilisant le service WAP de votre téléphone mobile !

Votre compte email supporte le protocole d'application Wireless, vous permettant de recevoir et d'envoyer des emails via votre téléphone mobile. Connectez-vous pour avoir accès à votre compte mail, et utilisez toutes les options que vous avez l'habitude de paramétrer pour votre accès WAP !

C'est très simple ! Sous le menu 'services' de votre téléphone WAP, consultez le nom de domaine de votre compte email: http://atmail.nt/wap.php (utilisez le nom de domaine de votre serveur). Ensuite, connectez-vous simplement en entrant votre login et votre mot de passe. A travers votre téléphone mobile, vous pouvez accéder à vos emails. Le WebMail WAP inclut 5 fonctions:

  • Montrer les messages non lus
    Cette fonction ouvre votre boîte aux lettres et != montre que les messages non lus. Utile pour accéder rapidement aux derniers messages de votre dossier de réception.
  • Montrer tous les messages
    Affiche tous les messages de la boîte de réception. A travers le panneau des paramètres, vous pouvez spécifier l'ordre et le formattage des messages qui seront affichés.
  • Composer un message
    Compose un message email depuis le clavier de votre téléphone; envoyez-le à n'importe quelle adresse email Internet.
  • Options
    Spécifiez les options du compte. Ce choix inclut le format du message, le style et les préférences de l'interface.
  • Carnet d'adresses
    Conserve les informations du carnet d'adresses pour un usage ultérieur. Utilisez votre carnet d'adresses pour conserver les comptes emails et les ajouter rapidement aux messages que vous composez.

Accédez à votre email de n'importe où, n'importe quand ! Pour plus d'informations, ou si vous rencontrez des difficultés, contactez votre fournisseur de messagerie.
