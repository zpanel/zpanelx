Changer les paramètres [ Index ]

Utilisez le panneau de configuration pour modifier les préférences de votre compte e-mail. Les préférences concernent:

Préférences du WebMail

  • Nom
    Le nom est utilisé lors de l'envoi d'un mail et sera affiché sur tous les emails sortants.
  • Répondre à
    Changez l'adresse de réponse si elle est différente de celle utilisée avec ce compte.
  • Trier les messages par
    Choisissez le critère de tri pour les messages dans votre boîte email.
  • Configurer l'interface
    Cochez les différentes fonctionnalités de l'interface du WebMail. Pour les anciens navigateurs, il est recommandé de faire simple. Vous avez la possibilité de désactiver les fenêtres 'popup' quand vous lisez les messages dans la boîte de réception.
  • Langue
    Séléctionnez la langue que vous désirez utiliser.
  • Messages à afficher
    Définissez le nombre de message à afficher dans la boîte de réception. Vous pouvez naviguer dans la boîte email avec les icones précédent/suite.
  • Taux de rafraîchissement du dossier des messages reçus
    Sélectionnez le nombre de minutes pour le rafraîchissement automatique du dossier des messages reçus. Vous serez avertis d'un nouvel email par un son.
  • Format du temps
    Indiquez votre préférence pour l'affichage du temps dans l'interface de Webmail et de Calendrier
  • Format de la date
    Sélectionnez le format d'affichage de la date
  • Choisir la zone horaire
    Entrez votre zone horaire. Quand vous envoyez des messages, ceci détermine l'heure et la date.
  • Signature de messagerie
    Entrez la signature pour vos messages. Elle sera appliqué à tous les emails sortants.
  • Vider la poubelle à la sortie
    Cochez cette option si vous désirez effacer le contenu de la poubelle quand vous vous déconnectez.
  • Editeur HTML
    Choisissez si vous voulez utiliser ou non l'éditeur HTML.
  • Signature d'email
    Indiquez votre signature. Elle sera incluse à tous vos emails sortants
