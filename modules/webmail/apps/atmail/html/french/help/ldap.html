Moteur de recherche LDAP [ Index ]

Utilisez le moteur LDAP pour chercher les addresses email, postale et le numéro de téléphone d'un utilisateur. LDAP est une base de données publique avec des coordonnées de gens du monde entier. Pour commencer une recherche vous devez indiquer le nom complet de la personne recherchée et en option son adresse email.

Quand vous consultez les résultats de la recherche, vous pouvez sélectionner des adresses et les rajouter dans le carnet d'adresses personnel, composer un message ou cliquer pour voir les informations supplémentaires concernant la personne sélectionnée.

Si l'option est activée pour votre service email, vous pouvez effectuer une recherche par rapport aux utilisateurs de votre domaine ou site en cliquant sur 'Type de recherche' ou 'Recherche dans la base de données du système'.
