Trucs et astuces du WebMail
  • Améliorez la performance de l\'interface WebMail en configurant votre navigateur pour sauvegarder en cache les pages HTML/images. Sur Internet Explorer, allez sur Outils -> Options Internet -> Paramètres, et cochez "Vérifiez s\'il existe une version plus récente Automatiquement".
  • Vous pouvez ouvrir un menu contextuel en cliquant avec le bouton droit de la souris sur un message. Il est ainsi plus facile de Répondre, Transférer ou d\'Ouvrir un message.
  • Vous n\'arrivez pas à vous rappeler les destinataires d\'un emails ? Facile ! Vous pouvez ajouter les destinataires directement depuis le carnet d\'adresses avec le bouton "Ajouter des destinataires". Vous pouvez même parcourir à la suite le carnet d\'adresses Global et Personnel !
  • Vous souhaitez utiliser des addresses sur d\'autres serveurs LDAP ? Pas de problèmes - Vous pouvez le faire dans la fenêtre Trouver des personnes où vous pouvez spécifier le serveur LDAP sur lequel vous souhaitez effectuer une requête, ainsi que les paramètres qui délimiteront votre recherche.
  • Saviez-vous que vous pouvez glisser-déplacer des messages d\'un dossier dans l\'autre. Pour ce faire, déplacez simplement un email sur le dossier de destination - c\'est un moyen plus rapide de déplacer vos messages.
