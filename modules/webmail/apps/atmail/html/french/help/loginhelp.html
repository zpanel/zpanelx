Aide au login du compte:

Pour aller sur votre compte $ref['brandname'], ou pour accéder à un compte externe en utilisant le client $pref['brandname'], vous aurez besoin de fournir un nom d'utilisateur et un mot de passe, de sélectionner le type d'interface, et votre préférence de langue. A titre optionnel, si vous utilisez $pref['brandname'] pour accéder à des messages externes, vous aurez besoin de fournir le nom du serveur de messagerie.

Nom d'utilisateur: c'est principalement un identifiant personnel avec votre serveur de messagerie, par exemple si l'utilisateur Jean a un compte avec isp.com, son adresse email sera: jean@isp.com.

Mot de passe: une clé secrète qui vous identifie pour entrer sur le compte de messagerie correspondant. Si vous oubliez votre mot de passe, ou que vous souhaitez le changer, cliquez sur le lien "Changer le mot de passe" dans la page de login

Avec vos comptes IMAP, votre session est dupliquée sur le serveur IMAP. Tous les dossiers que vous créez, ou les messages déplacés seront réfléchis sur le serveur IMAP distant. Tous les messages sont conservés sur la machine distante, ce qui vous permet d'accéder également aux données à partir d'un autre client email.

Langage: Vous pouvez sélectionner la langue d'utilisation du webmail.

Dépannage - Je != peux pas me connecter à mon compte:

Vérifiez que vous avez entré un mot de passe correct, faites appel à l'utilitaire des mots de passe si vous avez oublié ou si vous voulez changer votre mot de passe.
