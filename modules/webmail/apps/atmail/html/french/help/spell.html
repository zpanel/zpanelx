Particularité de la correction orthographique [ Index ]

En utilisant la correction orthographique, vous pouvez passer en revue votre message pour contrôler les erreurs. Si le mot n'est pas trouvé dans la base de donnée, le système vous suggère une liste. Choisissez un mot dans cette liste et cliquez le bouton 'Changer'. Si le mot n'est pas dans la liste, Vous pouvez entrer le votre dans le champs proposé.

Si le mot est othographié correctement, cliquez 'Ajouter personnel' pour le mot que vous souhaitez ajouter dans votre dictionnaire personnel. Pour ignorer une erreur d'orthograhe, cliquez sur 'Ignorer'.

La page retournera à la fenêtre de composition s'il n'y a pas d'erreurs dans votre email.
