FAQ LDAP [ Help Index ]

Comment est-ce que je fais une requête sur un serveur LDAP ?

Pour effectuer une recherche sur un serveur LDAP, faites ce qui suit:

  1. Aller sur Chercher > Carnet d'adresses
  2. Dans le menu déroulant indiqué "Type de recherche", sélectionnez "serveur LDAP"
  3. Indiquez les paramètres de votre serveur LDAP ci-dessous
  4. Remplissez les champs de recherche, ensuite cliquez sur "Recherchez dans la base de données"

Comment indiquer un serveur LDAP externe ?

Pour indiquer un serveur LDAP externe, remplissez le champs dans votre fenêtre de requête appelée "Serveur LDAP". Remplissez les champs de recherche comme vous le feriez avec une requête normale, ensuite cliquez sur "Faire une requête sur la base de données"
