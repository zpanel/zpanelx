Options de la boîte email [ Index ]

Cette rubrique vous permet de personnaliser les options pour votre boîte email et ses répertoires.

Dans la rubrique boîte email active, il y a une liste de tous les répertoires, le nombre de messages qui y sont conservés, la taille du répertoire et l'option permettant de supprimer ce répertoire.

La rubrique créer une nouvelle boîte aux lettres vous permet de rajouter un nouveau répertoire.

La rubrique bloquer des adresses email vous permet de refuser les message envoyés à partir d'une adresse particulière. Vous devez simplement ajouter l'adresse qui vous envoie du courrier indésirable ('spam') ici et cliquer sur Ajouter email.

L'option ajouter un message de réponse automatique va répondre de manière autonome à TOUS les emails entrants. Ajoutez le message que vous voulez voir utilisé pour la réponse automatique.

L'option trier les messages vous permet de configurer le WebMail d'un façon telle que les messages entrants soient automatiquement distribués dans des répertoires selon vos propres critères.
