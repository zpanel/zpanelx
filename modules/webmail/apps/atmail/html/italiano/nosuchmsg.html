Message not found

The email message cannot be loaded. Please reload your mailbox window to refresh the correct email listing from the server. This error can occur if:
  • The message no longer exists
  • The message has moved into another folder
  • The message has been removed from another application via POP3/IMAP
  • The message folder has been renamed since the email was last opened

Se l'accesso al tuo account non funziona ancora

  • Contatta l'amministratore $this->admin_email per assistenza
