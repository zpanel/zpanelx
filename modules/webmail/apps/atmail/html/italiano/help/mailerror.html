Errore Mail [ Indice Aiuto ]

Ricevo il seguente errore - errore di connessione al server – Il dominio esiste ed accetta connessioni in ingresso?

Questo è causato da un non corretto mailserver. Per risolvere fare quanto segue:

  1. Clicca sul link "Login Esterno" nella pagina di login
  2. Nella pagina di login che apparirà, inserire l’indirizzo completo del Mailserver che ospita la tua e-mail. Per esempio mail.yourdomain.com, pop3.yourdomain.com, mail.ispserver.com, etc. Trova il corretto indirizzo del mail-server controllando le impostazioni del tuo client desktop o contatta il tuo ISP per maggiori dettagli.
  3. Inserisci il tuo username e la tua password
  4. Clicca su "Login"

Ricevo il seguente errore quando accedo via Outlook - Trasmissione non permessa.

Per risolvere questo, segui le seguenti istruzioni:

  1. Apri Outlook
  2. Clicca su Strumenti > Accounts
  3. Clicca sulla scheda "Mail".
  4. Scegli il tuo account e clicca sulla scheda "Server"

Qui le informazioni del server dovrebbero essere come di seguito:

In entrata: pop.mydomain.com
SMTP: smtp.mydomain.com

Nome account: inserire l’indirizzo e-mail completo compreso il dominio

Quando accedo, ricevo un errore simile - "Can't open file: 'sometable.MYD'"

E’ possibile che il database nel tuo server sia corrotto. Per assistenza contatta l’amministratore e richiedi la riparazione del database.
