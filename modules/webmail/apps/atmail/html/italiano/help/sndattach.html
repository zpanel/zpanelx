Come si aggiunge un allegato ai messaggi?

Per allegare un file ad un messaggio segui le istruzioni:

  1. Prima clicca il pulsante Allegati nella schermata di composizione.
  2. Una nuova finestra pop-up permetterà di aggiungere allegati; clicca il pulsante "Sfoglia" per selezionare file dal computer
  3. Clicca "OK" quando hai selezionati gli allegati corretti.
  4. Una volta che i file sono stati caricati si potrà controllarne la dimensione o scegliere di cancellarli.

NOTA: Quando alleghi un file nel tuo messaggio e-mail, potrebbe aumentare in dimensione. Gli allegati inviati via e-mail devono essere modificati in un formato diverso per essere letti dagli altri programmi. Questo tipo di trasformazione aumenta la dimensione originale del 33% circa. Di conseguenza, un file di 2MB allegato potrebbe risultare di circa 2.66MB in formato e-mail.
