Assistenza Login [ Help Index ]

Assistenza account login:

Per entrare nel tuo account @Mail, o per accedere ad un account esterno usando @Mail, devi inserire un Username e una password, e selezionare il tipo di interfaccia e la lingua preferita. Puoi anche inserire un mail server per accedere ad account esterni a @Mail. (Opzionale)

Username: questo è comunemente l’identificazione personale nel vostro server e-mail, per esempio l’user, Jim ha un account e-mail con isp.com, il suo indirizzo e-mail è: jim@isp.com

Password: una chiave segreta che permette di accedere all’account e-mail. Se hai perso la password, o desideri cambiarla clicca sul link 'Cambia Password' nella schermata di login

Con un account IMAP , la tua sessione è mirrored direttamente dal server IMAP. Ogni cartella creata, o messaggio spostato avrà effetto anche sul server IMAP remoto. Tutti i messaggi rimarranno sul server remoto e saranno disponibili per altri accessi da altri client e-mail.

Lingua: Puoi selezionare la lingua con cui usare la WebMail.

Assistenza – Non posso fare il login Account:

Controlla di aver inserito la password correttamente prima di eseguire la procedura di recupero della password dimenticata o di modifica della stessa.
