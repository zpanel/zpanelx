WebMail Trucchi e Suggerimenti
  • Aumenta le performance della WebMail configurando il browser a salvare le cache di HTML/immagini. Visita le Opzioni Internet -> File Temporanei -> Impostazioni -> Seleziona per ricerca versioni più recenti delle pagine = "Automaticamente" .
  • Puoi aprire un menù contestuale cliccando il tasto destro del mouse sul messaggio. Così è molto più facile Rispondere, inoltrare, o Aprire un messaggio.
  • Non puoi ricordare un indirizzo e-mail per un destinatario? Semplice – puoi aggiungere il destinatario direttamente dalla rubrica con il pulsante "Aggiungi Destinatario". Si può anche ricercare nella rubrica globale
  • Vuoi usare gli indirizzi esistenti su altr server LDAP? Nessun problema – si può fare nella finestra Cerca Contatti, in cui si può specificare il server LDAP che si desidera usare, con le informazioni necessaria per limitare la ricerca a quello che si desidera.
  • Sai che puoi trascinare i messaggi con il mouse da una cartella ad un altra? Questo è un modo molto più semplice di spostare i messaggi.
