Guida Import della rubrica [ Indice Aiuto ]

queste informazioni ti aiuteranno nell’importare una rubrica esistente nella tua WebMail.

  • Seleziona impostazioni di importazione
  • Seleziona i campi corrispondenti

Seleziona le opzioni di importazione 

A per importare un indirizzo hai bisogno della data del file CSV. CSV i valori sono separati da virgola, e simili a:

Nome, Cognome, Secondo Nome, Indirizzo e-mail, Via,Città,....

Se importi la tua rubrica da un altro client come Outlook, puoi esportare i dati in formato CSV con il programma.

Per esempio se vuoi importare un contatto dalla rubrica di Outlook Express, devi seguire i seguenti passaggi.

    1: In Outlook, clicca su File --> Esporta --> Rubrica.

    2: Allora avrete l'opzione per selezionare il formato per l’esportazione.

    Seleziona "Testo file (Separato da una virgola)"

    3: Avrete salvato il file CSV in qualche parte nel vostro computer. Seleziona il pulsante sfoglia per ritrovarlo.

    4: Ora è possibile selezionare gli indirizzi dalla vostra rubrica da imposrtare nella WebMail. Clicca "Fine" quando hai fatto.

    5: la tua rubrica CSV dovrebbe essere importato correttamente. Ora ritorna all’interfaccia WebMail.

    Visitare le funzioni della Rubrica e selezionare il pulsante "Importa". Inserire le informazioni necessarie.

La prima riga continie la descrizione del file CSV: Molti file CSV, inclusi i file creati con Outlook, contengono una colonna descrizione. Se non sei sicuro, puoi aprire il file CSV attraverso Notepad.

Clicca sul pulsante "sfoglia" dalla pagina di importazione della rubrica e seleziona il CSV che hai creato. Successivamente clicca "Importa File" ed è fatto!

Seleziona Campo Corrispondente [alto]

Qui puoi selezionare i campi dal tuo CSV che corrispondono con la rubrica della WebMail.

WebMail cercherà di trovare i risultati migliori, tuttavia se non fossero corretti puoi modificare i parametri attraverso il menù a discesa.

Se c’è un campo con "Seleziona" significa che WebMail non ha trovato i parametri automaticamente. Ciò può dipendere dal fatto che non sono stati inclusi nel file CSV. Questo non influenzerà l’operazione di esportazione e non causerà problemi finchè il campo indirizzo e-mail è correttamente collegato.

Una volta fatto, clicca su "Importa".

Se l’operazione andrà a buon fine sarete avvisati del numero di contatti e quindi apparirà la rubrica aggiornata.
