Message cannot be moved

The unique message identifier does not match the selected message for move/deletion. This error can occur if:
  • You have refreshed the WebMail window after moving a message. The previous message cannot be moved twice
  • The message has been moved from another application via POP3/IMAP. Please reload your mailbox folder for an updated listing
  • The message no longer exists on the mail-server
