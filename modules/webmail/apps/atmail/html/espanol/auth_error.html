Error de Acceso

Acceso Denegado - Tu SessionID podría no estar validada en la base de datos
  • Verifica que tu Nombre de Usuario y contraseña sean correctas.
  • Verifíca que tengas habilitada la opción de aceptar Cookies
  • Verifíca que no tengas activas dos secciones al mismo tiempo - sólo puedes tener una por Nombre de Usuario

Si aún no has logrado ingresar a tu Cuenta de email

  • Utiliza la Utilidad de contraseña para recuperar contraseñas perdidas
  • Contacta con el Administrador $this->admin_email para recibir ayuda
  • Crear una Nueva Cuenta de Correo
