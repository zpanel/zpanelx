Import AddressBook Help Guide [ Indice de ayuda ]

Information found here will assist you in importing an existing address book into your WebMail account.

  • Selecting Import options
  • Selecting corresponding fields

Selecting Import options 

To import an address book you will need the date to be in CSV file format. CSV stands for Comma Separated Values, and would look similar to:

First Name,Last Name,Name,E-mail Address,Home Street,Home City,....

If your importing your address book from another email client such as Outlook, you can export your address data to CSV within this program.

For example if you would like to import an address book from Outlook Express, you would take the following steps.

    1: In Outlook, visit File --> Export --> Address Book.

    2: You will then have the option to select what sort of format you would like the export to convert to.

    Select "Text File (Comma Separated Values)"

    3: You will then have the option to store the CSV file somewhere on your computer. Select the browser button to find a suitable place and name the file.

    4: You can now select which data from your address book you would like to import into your WebMail. Click "Finish" when your done.

    5: Your address book CSV should now be successfully completed. Now return to the WebMail interface.

    Visit the Address Book function and select the below button "Import". Fill in the necessary information.

First row contains CSV column description: Most CSV files, including files created via Outlook, contain the column descriptions. If your not sure, you can check if your CSV contains the headings by opening the file in Notepad.

Click on the "browse" button from the Address Import book and select the CSV file you created. Then click "Import File" and your almost there!

Selecting corresponding fields [top]

Here you can select which fields from your CSV match with the WebMail address book fields.

WebMail will try to find the best match, however if its not correct you can change it via each fields drop down menu.

If there is a field with "Select" it means that WebMail was unable to find data for this field. This may be because it was not included in your CSV. It won't affect the export or cause any problems, as long as the EMail address field is correctly linked up.

Once your done, click "Export".

If successful you will be notified of the number of new contacts, then be directed to your newly updated Address book.
