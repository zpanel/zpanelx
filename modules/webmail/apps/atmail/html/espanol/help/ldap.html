Motor de búsqueda LDAP [ Indice de ayuda ]

Utiliza el motor de búsqueda LDAP para encontrar usuarios de email, direcciones y números de teléfono. LDAP es una base de datos pública con información sobre usuarios de todo el mundo. Para comenzar una búsqueda debes especificar el nombre completo del usuario que estás buscando, opcinalmente puedes introducir también su dirección de email.

Cuando obtengas el resultado de tu búsqueda puedes optar por añadir el usuario en tu libreta de direcciones, escribirle un email u obtener más información sobre el usuario.

Si esta disponible es su servicio de correo, usted puede realizar una busqueda en su dominio seleccionando 'Tipo de Busqueda'' como 'Base de Datos del Sistema'.
