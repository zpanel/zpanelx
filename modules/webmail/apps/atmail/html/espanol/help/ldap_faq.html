LDAP FAQ [ Help Index ]

How do I query an LDAP server?

To query an LDAP server, do the following:

  1. Go to Search > Addressbook
  2. In the drop-drown box labeled "Search Type", select "LDAP Server"
  3. Specify your LDAP server details below
  4. Fill in the search fields, then click on "Query Database"

How do I specify an external LDAP server?

To specify an external LDAP server, fill in the text field in your query window called "LDAP server". Fill in the search fields as you would do with a normal query, then click on "Query Database"
