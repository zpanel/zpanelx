Отправка почты [Наверх]
 Через учетную запись Веб Почты, можно отправить письмо на любой активный почтовый ящик в Интернете. С мощным HTML-редактором (только в IE4/IE5) можно создавать навороченное оформление HTML писем, или, для совместимости (простой текст) используйте встроенный редактор обычного текста.

Как послать почту

  1. Войдите в свой ящик.
  2. Щелкните Написать письмо в панели навигации
  3. Введите информацию в заголовке:
    • Кому: Введите получателей письма, или выберите из адресной книги. Нескольких получателей введите через запятую.
    • Тема: Введите тему письма.
    • Копия: Введите адреса тех кто также должен получить письмо. Копия письма будет им отослана.
    • Скрытая копия: Введите адреса тех, кому вы также хотите отправить копию письма, но при этом чтобы другие получатели этого не увидели.
  4. Напишите текст письма в специально предоставленной для этого области.
  5. Если нужно, щелкните кнопку Прикрепить файлы чтобы послать с письмом файлы.
  6. Щелкните кнопку Отправить Почту.

Особенности Исходящих писем [Наверх]

  • Прикрепленные файлы
    Чтобы прикрепить файлы к письму, сначала щелкните кнопку Прикрепить файлы в окне создания письма. Откроется новое всплывающее окно; щелкните "Обзор" для выбора файла с жесткого диска. Щелкните "OK" когда вы выберет нужный файл. После того как файл закачается, вы можете проверить размер прикрепленных файлов и удалить если надо. Можно прикрепиь много файлов; по соображениям безопасности и квотам на сервере прикрепления не могут занимать более &lt;2-10 Мегабайт. Для точных размеров ограничения обратитесь к Администратору.

  • Подписи
    При отправке письмо может быть подписано. См. панель настройки для дополнительной информации.
