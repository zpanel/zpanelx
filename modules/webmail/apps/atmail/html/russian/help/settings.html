Изменение Параметров настройки [ Индекс справки ]

Используйте панель параметров настройки, чтобы изменить предпочтение вашей учетной записи электронной почты. Настройте учетную запись на свои потребности, включая:

Настройки Почты

  • Полное имя
    Это используется при создании электронной почты, ваше Полное имя будет показываться во всех исходящих письмах.
  • Ответ на адрес
    Введите ваш ReplyTo адрес электронной почты если он отличается от вашего имени входящего в системе.
  • Сортировка писем
    Укажите порядок сортировки писем в папках.
  • Укажите вид
    Выбирайте из различных интерфейсов Веб Почты. Рекомендуется Простой рекомендуется для старых браузеров, или если вы хотите отключить всплывающие окна при просмотре писем.
  • Язык системы
    Укажите язык для использования в качестве основного в интерфейсе почты.
  • Количество показываемых Писем
    Укажите число писем для одновременного показа в папке Входящие. Пользуясь значками вперед/назад можно смотреть разные письма.
  • Интервал запроса новой почты
    Укажите интервал в минутах для запроса новой почты. При получении новых сообщений вы услышите звуковой сигнал.
  • Формат времени
    Выберите формат временя для интерфейса Почты и Календаря
  • Формат даты
    Выберите формат для показа даты
  • Часовой пояс
    Укажите часовой пояс где вы сейчас находитесь. Используется при отправке писем.
  • Подпись писем
    Введите вашу подпись электронной почты. Она будет добавлена в конец ко всем исходящим письмам.
  • Очищать корзину при выходе
    Переключите, если Вы желаете очистить Корзину выходя из системы.
  • HTML редактор
    Выберите, если Вы хотите использовать редактор HTML.
  • Подпись
    Создайте вашу подпись. Она будет присоединятся к любым отправленным вами письмам.
