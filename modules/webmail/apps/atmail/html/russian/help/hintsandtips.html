Совет дня
  • Затруднительно будет пользоваться интерфейсом WebMail, если ваш браузер настроен на кэширование изображение и HTML. Настройте ваш браузер: Сервис -> Свойства Обозревателя -> Временные файлы Интернет - Параметры -> = "Автоматически".
  • Вы можете использовать контестное меню, нажав правую кнопку мыши при чтении и работе с письмами. Это самый короткий путь, если нужно Ответить, Переслать или Открыть письмо..
  • Не можете запомнить всех адресатов для вашей почты? Вы легко можете добавить нового получателя используя кнопку Добавить получателя.
  • Ищите адресата на других LDAP серверах? Нет проблем - используйте Поиск по LDAP северам.
  • Знаете ли вы что можно использовать click and drag для перемещения писем из одной папки в другую?.
