Поисковый сервер LDAP [ Индекс справки ]

Используйте LDAP поисковый сервер, чтобы найти электронные адреса пользователей, адрес и телефонные номера. LDAP - общественная база данных информации о людях по всему миру. Чтобы начинать поиск, Вы должны определить полное имя пользователя, и необязательный адрес электронной почты.

При рассмотрении результатов поиска Вы можете выбирать - добавить пользователя к вашему списку контактов, написать письмо или щелкнуть для дополнительной информации относительно пользователя.

Если разрешено Вы можете искать в вашем домене, сменив ' Тип Поиска ' на ' Искать в Системной Базе данных '.
